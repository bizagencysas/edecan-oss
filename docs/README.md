# docs

Documentación extendida del proyecto, en español. Empieza por [`index.md`](./index.md) — es el mapa completo con enlace y descripción de cada documento:

- Self-hosting, la app de escritorio y configuración (instalación, variables de entorno, credenciales bring-your-own: LLM por API key/CLI local/Ollama/Vertex, voz, conectores).
- Referencia de la API HTTP y de la personalización "nivel Dios" (persona + memoria).
- Catálogo de conectores oficiales soportados y cómo registrar tu propia app OAuth por proveedor.
- Voz web vs. telefonía: cumplimiento de consentimiento, ventanas horarias, opt-out ("STOP"/"BAJA"), identificación como asistente automatizado.
- Cumplimiento: privacidad (GDPR/CCPA/LFPDPPP/Ley 1581) y términos de servicio de cada red.
- Modelo de amenazas y runbooks de incidentes (fuga entre tenants, rotación de claves, restauración de base de datos, cola atascada).
- El resto del ecosistema del producto (negocio/inventario/RRHH/ads/viajes, ecosistema de agentes/automatizaciones/misiones, analista, reuniones, MCP, apps móviles, casa inteligente) — ver `index.md` para el mapa completo por ola.
- Roadmap de producto.

Para la arquitectura técnica y los contratos vinculantes, ver [`../ARCHITECTURE.md`](../ARCHITECTURE.md); para la evolución pública del producto, [`roadmap.md`](./roadmap.md).
